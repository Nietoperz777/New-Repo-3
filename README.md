<b>Jak dodać mięsko</b>. Krok pierwszy klikamy liczbę podaną w nawiasie (1). 
Krok drugi wybieramy ID do którego jest przypisane odpowiednie mięsko które chcemy wybrać by zaspokoić nasz apetyt. 
Krok trzeci wybieramy jak chcemy nazwać nasze mięsko zanim je zdegustujemy.
Krok czwarty wybieramy dla naszego mięska specjalną kategorie przypisaną: (od 0 do 5) która doda mu lepszego smaku. W magazynie mamy jeszcze trzynacie lub czternaście opcji czekającyb na dofinansowanie a tak to przykro ale trzeba się zadowolić tym co jest przepraszamy bardzo szanownych klientów w razie zabrakło więcej różnorodnych smaków do spróbowania. Obiecujemy że za jakiś czas dodamy więcej prosimy o cierpliwe oczekiwanie na dalszy postęp w tym przeziewizieciu.
